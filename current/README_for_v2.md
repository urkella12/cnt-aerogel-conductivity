## Файлы
- `ФИНАЛ.py` — основная программа с исправлениями проводимости матрицы

## Изменения v2.1
- Добавлен расчёт проводимости матрицы (σ = 1×10⁻⁶ См/м при φ < φ_c)
- Добавлен вывод объёмных долей (φ_УНТ, φ_аэрогель, φ_total)
- Исправлена ошибка: σ = 0 → σ = 1e-6 для непроводящих систем

## Известные ограничения
- Модель работает в "игрушечном" масштабе (L/L_field ~ 0.5 vs реальные 0.001)
- Finite-size эффекты требуют калибровки для макроскопических систем
- Применима для качественного анализа, не для прямого сравнения с экспериментом
