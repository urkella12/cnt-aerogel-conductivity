## 📋 Описание

Вычислительная модель для исследования электропроводящих свойств композитов с углеродными нанотрубками (УНТ). 

**Методы:**
- 3D геометрическое моделирование структуры
- Теория перколяции (graph theory)
- Метод Кирхгофа для расчёта проводимости
- Туннелирование между УНТ
- Статистический анализ (Монте-Карло)

**Результаты:**
- Порог перколяции: **φ_c = 0.27 ± 0.07%**
- Степенной закон: **σ ∝ (φ - φ_c)^1.86** (R² = 0.9989)
- Калибровка на 250+ симуляциях

## 📊 Основные результаты

### 1. Порог перколяции
```
φ_c = 0.27 ± 0.07 об.%
```
Согласуется с литературой для aligned УНТ (0.1-0.5%).

### 2. Степенной закон
```
σ = σ₀ · (φ - φ_c)^t
t = 1.86 ± 0.05  (R² = 0.9989)
```
Близко к теоретическому t ≈ 2.0 для 3D перколяции.

### 3. Зависимость от длины
```
σ ∝ exp(α·L)
α ≈ 0.011 нм⁻¹  (для L = 300-500 нм)
```
Рост в **10 раз** при увеличении длины на 200 нм.

### 4. Влияние ориентации
- **jitter = 0°**: Нет перколяции
- **jitter = 4-8°**: Резкий скачок σ
- **jitter = 15-30°**: Оптимум
- **jitter > 30°**: Плато

## 🎓 Область применимости

### ✅ Применима для:
- Композитов с плотной матрицей (ρ > 0.2 г/см³)
- Объёмных долей φ = 0.1-5%
- Длин трубок L = 100-500 нм
- Контролируемой ориентации УНТ

### ❌ Не применима для:
- Аэрогелей (пористость >90%)
- Макроскопических образцов (>10 мкм)
- Очень длинных УНТ (L >> 1 мкм)

## 📖 Методология

1. **Генерация геометрии**
   - Случайное распределение центров УНТ
   - Контроль ориентации (aligned/random + jitter)
   - Проверка минимальных расстояний

2. **Вокселизация**
   - Дискретизация 3D пространства (типично 200³)
   - Разрешение ~2 нм

3. **Анализ перколяции**
   - Построение графа контактов (NetworkX)
   - Учёт туннелирования: G = G₀·exp(-β·d)
   - Поиск связного пути

4. **Расчёт проводимости**
   - Метод Кирхгофа: ∑I = 0 в узлах
   - Решение системы линейных уравнений
   - Анизотропный расчёт (3 направления)
